ELITE is an advanced AI-powered legal assistant chatbot that provides quick, reliable, and accessible legal advice. Whether you need to understand your rights, draft basic legal documents, or get instant guidance on common legal issues — ELITE is available 24/7 to help you make informed legal decisions.

⚖️ Features
✅ Instant Legal Advice — Get AI-powered answers to your legal questions.

📄 Document Assistance — Basic help with drafting and understanding legal documents.

🕒 24/7 Availability — Always ready to assist, anytime, anywhere.

🔒 User-Friendly & Secure — Simple, safe, and easy to use.

🌐 Multi-Domain Knowledge — Covers a wide range of legal topics.

🛠️ Technologies Used
React.js — Frontend framework for building the user interface.

Node.js & Express — Backend services and API endpoints.

MongoDB — Database for storing user queries and logs.

OpenAI / Hugging Face — Powering the chatbot’s natural language understanding.

Tailwind CSS / SCSS — For clean, responsive styling.
